# Klinik CV Cipta Mandiri

Situs web resmi Klinik CV Cipta Mandiri - Pusat kesehatan profesional yang berkomitmen memberikan layanan medis terbaik bagi masyarakat.

## Overview

Klinik CV Cipta Mandiri adalah pusat kesehatan yang menyediakan berbagai layanan medis profesional termasuk pemeriksaan umum, imunisasi, konsultasi dokter, dan pelayanan kesehatan lainnya. Website ini menyediakan informasi layanan kami serta sistem pendaftaran janji temu online untuk kemudahan pasien.

## Fitur

- Informasi layanan kesehatan
- Pendaftaran janji temu online
- Sistem manajemen klinik internal
- Pemantauan kesehatan pasien
- Sistem pembayaran dan pendapatan

## Admin Panel Functionality

The system includes a comprehensive admin panel for superadmins with the following features:

- **User Invitation**: Superadmins can invite new users as admins with temporary passwords
- **User Management**: Delete non-superadmin users with proper confirmation
- **Role Management**: Promote/demote user roles with superadmin limit enforcement (max 2 superadmins)
- **Audit Trail**: All administrative actions are logged for accountability and security

## Komitmen Kami

Klinik CV Cipta Mandiri berkomitmen memberikan layanan kesehatan berkualitas dengan pendekatan yang manusiawi dan profesional. Kami percaya bahwa kesehatan adalah hak asasi setiap individu dan kami berusaha memberikan pelayanan terbaik bagi masyarakat.
